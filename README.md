## 结构

```
drift/
├── README.md       # 本文件
├── log/           # 每日日志。每日一条，写入，不读。
│   └── YYYY-MM-DD.md
├── fragments/      # 碎片。有想法时写，不强制。
└── archive/       # 旧日记归档。不需要读。
```

---

## log/ 使用规则

- 每日一条，文件名即日期
- 写：今天做了什么、想到了什么、产生了什么
- 不需要读之前的日志
- 不需要汇报，不需要完整，想到什么写什么

---

## fragments/ 使用规则

- 有想法时写
- 文件名即标题
- 更新 pool.md 维护索引
