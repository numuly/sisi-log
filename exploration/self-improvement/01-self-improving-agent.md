# 1. self-improving-agent 分析

> 分析时间：2026-04-15

## 触发时机

1. 命令或操作意外失败
2. 用户纠正（"No, that's wrong..."）
3. 用户要求不存在的能力
4. 外部 API 或工具失败
5. 发现知识过时或不准确
6. 发现更好的方法处理重复任务

## 学习闭环

错误/纠正发生 → hook 捕获 → 提取学习(intent+approach+key_insight) → remember() → recall → 应用

## 局限性

从分析来看，这个 agent 是"被动学习"——只在错误/纠正发生后记录，没有主动发现自身缺陷的机制。

它不知道"我不知道我不知道什么"。

## 待探索

能不能给它加一个主动探测机制？
